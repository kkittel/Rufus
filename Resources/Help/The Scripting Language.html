The Scripting Language

    Rufus has a primitive built-in scripting language. It is incomplete and doesn't work very well. You will find example scripts in the data folder. The script names must end in ".scr". You can use a text editor to view the example scripts or create new ones. To run a script in the data folder, say "run script <scriptname>" to Rufus. Also, you can create any phrase to run a script. Look up the phrase "knock knock" for an example. If you type "knock knock" into Rufus, you can tell him a knock knock joke and he will laugh.

This is an example of a simple sript file:

echo whos there
wait for input suppress actions
<name> = <object1>
echo <name> who
wait for input suppress actions
echo Ha Ha Ha!!!

echo repeats the text to the screen.
wait for input spress actions - waits for input and does not process it as it normally would
<name> = <object1> - assignment of variable <name> to <object1>

So this script lets you tell Rufus a knock knock joke and then he will laugh.

To fully understand or modify the scripting language that Rufus uses, see the scripts.m file.
