How it Works

    Rufus uses pattern matching, logic, memory, substitution, and searching to achieve its Artificial Intelligence. At the heart of the pattern matching is the concept of <object> phrase <object>, where <object> can be one or more words, or even another sentence altogether. Phrase is a fixed phrase. The phrases which the program uses to do its pattern matching are kept in a table, or file. For instance, a phrase might be <object> is <object>. So if the input to the program is "The ball is yellow" it would match that as <the ball> is <yellow>.  Object1 is <the ball> and object2 is <yellow>. The action for this phrase tells the program to store this phrase in temperary memory.
    Next, we could tell the Snlup that "Yellow is a color". It would match that as <yellow> is <a color> and store it in memory. Then if we ask it "What color is the ball" it would match as what <color> is <the ball>. It will use substitution to replace <color> with <yellow> and upon searching memory for <the ball> and <yellow>, will find the fact that has been stored, "The ball is yellow".
